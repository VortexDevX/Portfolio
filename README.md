# VAIBHAV PATEL // PORTFOLIO_

A high-performance, WebGL-accelerated 3D developer portfolio designed with a brutalist, cinematic terminal aesthetic. Built to showcase full-stack engineering capabilities through immersive, interactive data nodes rather than standard static web pages.

### ARCHITECTURE_
- **React 18** & **Vite** — Core UI construction and hot-reload build pipeline.
- **Three.js** & **React Three Fiber (R3F)** — GPU-accelerated 3D rendering and mathematical camera interpolation.
- **Tailwind CSS** — Strict utility-first brutalist styling matrix and complex global responsive viewport scaling.
- **Zustand** — Lightweight global state management synchronizing strict DOM overlays seamlessly with 3D canvas coordinates.

### EXECUTION_PROTOCOLS
Clone the repository and initialize the local development server:
```bash
npm install
npm run dev
```
