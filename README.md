# VAIBHAV PATEL // PORTFOLIO\_

A high-performance, WebGL-accelerated 3D developer portfolio with a brutalist, cinematic terminal aesthetic. Built to showcase full-stack engineering capabilities through immersive, interactive data nodes rather than standard static web pages. Dual-surface architecture: a GPU-rendered 3D scroll experience on desktop, and a purpose-built touch-first terminal data-pad on mobile.

---

### ARCHITECTURE\_

- **Next.js 15** & **React 19** — App Router, server components, edge-optimized deployment pipeline.
- **Three.js** & **React Three Fiber (R3F)** — GPU-accelerated 3D rendering with scroll-synced camera interpolation and texture-mapped project planes.
- **Tailwind CSS** — Strict utility-first brutalist styling with viewport-adaptive scaling across both surface layers.
- **Zustand** — Lightweight global state synchronizing DOM overlays with 3D canvas coordinates in real-time.
- **Web3Forms** — Zero-backend contact form integration with environment-secured access keys.

### DUAL_SURFACE_STRATEGY

| Surface     | Trigger                           | Experience                                                                                                                     |
| ----------- | --------------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| **Desktop** | `pointer: fine` / `width ≥ 768px` | Full WebGL 3D scene, scroll-controlled project planes, custom cursor, particle field, modal intelligence views                 |
| **Mobile**  | `width < 768px`                   | Zero WebGL. Purpose-built terminal data-pad with boot sequence, sticky HUD, decrypt-reveal project cards, touch-optimized form |
